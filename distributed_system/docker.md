
### Docker Architecture
- Docker 使用客户端-服务器 **(C/S)架构模式**，使用远程API来管理和创建Docker容器。
- Docker 容器通过 Docker 镜像来创建。
- 容器与镜像的关系类似于面向对象编程中的 **对象与类**。

| Docker | 面向对象 |
| :--- | :--- |
| Container  | 对象  |
| Image  | 类  |
